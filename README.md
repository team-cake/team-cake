### :arrow_down: Setup :arrow_down:

To run this Alex, download it using:  

```
$ npm install water
$ npm install food
$ npm install music
```

Or install it locally after sending an email:

```
$ cd ../mail
$ npm install mail
$ npx sequelize-cli generate:mail eka.cheuk@gmail.com
$ expo start sending mail
```
